ANTES DE COMENZAR A TRABAJAR EN EL PROYECTO
--------------------------------------------

Debemos sincronizar nuestra compu con GIT. Es decir descargar la 
última versión de proyecto a nuestra PC.

En la terminal, la ventana que está abajo del código, escribimos 
las siguientes sentencias:

git fetch  -->  NO SE QUE HACE
git pull origin main  -->  Descarga la última versión desde GITHUB

Ahora ya podemos arrancar.


ANTES DE TERMINAR DE TRABAJAR EN NUESTRO PROYECTO
--------------------------------------------------

Siempre debemos guardar nuestro trabajo en GITHUB, para que nuestros 
compañeros lo tengan disponible y arranquen desde ahi.
Para guardar los cambios debemos escribir en la terminal los 
siguientes códigos: 

git add .   -->  NO SE QUE HACE
git status   -->  para ver como va, y finalmente, 
git commit -m "texto explicativo de nuestro avance"  --> Le agrega un detalle de lo que hicimos
git status   -->  para ver como va, y finalmente, 
git push origin main  -->  para subirlo a github..


******* MUY IMPORTANTE  *******
Ponernos de acuerdo para no trabar de manera paralela en el 
mismo archivo. Ya que el segundo eliminará al primero.


PARA ELIMINAR LOS ÚLTIMOS CAMBIOS REALIZADOS SOBRE EL TRABAJO EN MI MÁQUINA
----------------------------------------------------------------------------

git reset --hard HEAD

Si realicé modificaciones que quiero descartar para 
oder descargar el trabajo actualizado del main.
